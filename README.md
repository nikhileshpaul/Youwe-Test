STEPS:

1. Use a fresh drupal9 setup
2. Install the **youwe_api** module using drush or from admin interface
3. Go to admin/structure/block and place the below blocks in the corresponding regions. Also, configure the blocks to be rendered only on <front> page and save them
4. Blocks should now be rendered in the front page
5. For Search Beer block, enter the name of food you would like to pair the beer with
6. For Search Cat Block, select the cat breed from the select list and submit to get the details
